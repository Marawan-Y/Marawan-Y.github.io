🤖 AGV · Real-Time Fleet
AGV RTLS Dashboard
Problem
No live visibility into AGV fleet positions, zone utilisation, or anomaly patterns inside a production facility.
Approach
Rebuilt a Python/Flask tracking system with MQTT ingestion at 3 Hz, WGS84 → UTM → affine coordinate transforms, MySQL persistence, and a Streamlit dashboard rendering live trajectories, heatmaps, and zone overlays. Docker-orchestrated broker, database, and UI services.

🔮 ML · Demand Forecasting
Material Consumption Forecast Dashboard
Problem
Material consumption per machine was unpredictable, with zero-consumption days inflating forecast errors when modelled with a single approach.
Approach
Designed a two-stage ML pipeline: a logistic classifier to flag zero-consumption days, followed by an LSTM regression model for magnitude prediction. Data is pulled and merged from Snowflake; modular codebase (fetch, prepare, forecast, train) is extensible to additional machines.
Snowflake Dash / Plotly LSTM Logistic Classifier Python
🧠 2-stage pipeline: zero-day detection + magnitude LSTM

⚑ Enterprise-safe: model architecture shared; training data and machine identifiers abstracted.

Schaeffler Mobility Insight Platform

How I Work

A consulting delivery method.

Six phases — from discovery to scale — designed for industrial environments where adoption matters as much as accuracy.

01 / DISCOVER
Discover
Stakeholder interviews, process mapping, pain-point prioritisation
02 / DATA
Data Readiness
Source audit, schema design, pipeline feasibility, data quality baseline
03 / MVP
MVP Analytics
First-pass dashboard or KPI report — validate value before model build
04 / MODEL
Model Build
ML pipeline design, evaluation gates, variance tracking, human review loops
05 / DEPLOY
Deploy & Monitor
Containerised deployment, operator-ready UI, drift monitoring in place
06 / SCALE
Scale
Modular extensibility, documentation handover, additional use-cases
Reliability over novelty
The right model is the one operators trust and use daily — not the latest architecture. Proven, monitored, explainable.
Adoption as engineering
UI design, export formats, and operator UX are first-class engineering concerns — not afterthoughts. Systems that aren't used don't deliver value.
Traceability by design
Every pipeline decision — data source, model choice, threshold — is logged, auditable, and reviewable. Especially critical for GenAI outputs.
